Cloud Support Engineer

Collaborate with engineering teams to evaluate and identify optimal cloud configurations and solutions
Design, develop, and deploy modular cloud-based systems and services
Ensure efficient functioning of data storage, processing, interconnectivity, and security in the cloud
Identify, analyze, and resolve infrastructure vulnerabilities and application deployment issues
Regularly review existing systems and make recommendations for improvements
Educate teams on the implementation of new cloud technologies and initiatives
Establish monitoring, maintain the same, and monitor for issues
Provide cloud support and make recommendations based on client needs
Implement and manage AWS cost optimization strategies
Automate cloud operations with robust notification and escalation processes
Ensure compliance with AWS security best practices and conduct regular security reviews


Skills Required:


Must have skills
Minimum of 3 years of experience as a Cloud Engineer or in a similar role
Proficiency in AWS, with relevant certifications preferred
Strong programming and scripting skills in languages like Python, Go, or Bash
Experience with infrastructure as code tools like Terraform or CloudFormation
Knowledge of containerization technologies like Docker and orchestration platforms such as Kubernetes
Understanding of cloud security best practices and ability to implement security controls
Excellent troubleshooting and problem-solving skills
Superior communication and collaboration skills to work effectively with cross-functional teams
Ability to mentor and guide more junior colleagues and others


Nice to have skills
AWS Cost Management and Optimization: Expertise in managing and optimizing AWS costs, including cost monitoring, resource allocation, and implementing cost-saving measures to achieve year-over-year cost reductions.
Automation and Scripting: Proficiency in automating cloud operations with full notifications and escalations using tools like AWS Lambda, CloudWatch, and other AWS automation services to enhance efficiency and reliability.
AWS Security and Compliance: In-depth knowledge of AWS security best practices, including identity and access management (IAM), encryption, and compliance frameworks to ensure robust cloud security and data protection.
